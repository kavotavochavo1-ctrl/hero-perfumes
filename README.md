# SILLAGE — Hero коллекции Noir

Премиальный hero-блок парфюмерного магазина на чистом HTML и CSS. Упор на тактильную карусель-coverflow и элегантные интеракции — без единой строки JavaScript.

## Ключевые особенности

Центр композиции — **coverflow-карусель из трёх флаконов**: крупный по центру и два уменьшенных размытых по бокам, переключаются кликом по флакону, стрелке или точке. Выбор флакона синхронно перелистывает заголовок, описание, ноты аромата и цену.

Вся интерактивность держится на состоянии CSS. Механизм: **«карусель целиком работает на трёх `<input type="radio">` и комбинаторе соседей `:checked ~ .aWrap` — смена флакона, кроссфейд контента, видимость стрелок и подсветка точек выводятся из того, какой radio выбран, без JS»**.

Группы контента стопкой через CSS Grid (`.swap`) и кроссфейд по `opacity` + `visibility`. Глубину задают пульсирующее радиальное свечение `glow`, тёплая световая «лужа» `pool` и контактная тень под активным флаконом. На hover: подъём переднего флакона, подсветка навигации, анимация стрелок и CTA.

## Доступность и адаптивность

Скрытые radio-инпуты остаются управляемыми с клавиатуры — Tab до группы, стрелки переключают флаконы нативно. У стрелок навигации — `aria-label`, у изображений товара — описательный `alt`, декоративная графика помечена `aria-hidden`. Вся анимация уважает `prefers-reduced-motion`.

Раскладка адаптируется на одном брейкпоинте: полный coverflow из трёх флаконов на десктопе сворачивается до одного центрального флакона на мобильном (до 640px), где навигация, боковые флаконы и детали trust-полосы скрыты ради фокуса на узком экране. Везде используется `clamp()` — типографика и отступы масштабируются без лишних брейкпоинтов.

## Технологии

HTML5 и CSS3 (Grid, Flexbox, `clamp()`, комбинаторы соседей, keyframe-анимация), zero JavaScript. Типографика — дисплейный сериф **Cormorant Garamond** в паре с гротеском **Jost**. Вся презентация в `styles.css`, разметка — только контент.
